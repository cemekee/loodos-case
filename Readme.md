# Loodos

Loodos is a movie search application.

## Installation

Loodos is available through CocoaPods. To install it, simply add the following line to your Podfile:

```bash
pod 'Kinfgfisher'
pod 'SwiftyJSON'
pod 'Alamofire'
pod 'Firebase'
```

Finaly

```bash
pod install
```
